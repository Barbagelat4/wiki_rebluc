# Capturas del Laboratorio

Esta carpeta contiene las capturas de pantalla del trabajo realizado en las máquinas virtuales.

## Estructura de Imágenes

### Capturas Requeridas:
- `inicio.png` - Captura de la topología o diagrama inicial
- `instalacion.png` - Configuración básica del servidor SRV-DC01
- `active_directory.png` - Promoción a DC y estructura de OUs
- `cliente_dominio.png` - Unión del cliente WIN10-CLIENT al dominio
- `servicios_red.png` - Configuración de DNS y DHCP
- `gpo.png` - Políticas de grupo implementadas

## Instrucciones para Agregar Imágenes

1. Toma las capturas de pantalla de tu trabajo en las máquinas virtuales
2. Guárdalas con los nombres indicados arriba
3. Cópialas también a la carpeta `public/img/` para que sean accesibles desde los componentes React
4. Los componentes ya están configurados para mostrar estas imágenes

## Formato Recomendado
- Formato: PNG o JPG
- Resolución: Mínimo 1024x768
- Calidad: Alta definición para que se vean claramente los detalles
