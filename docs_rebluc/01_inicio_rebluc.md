# 01 - Inicio y Portada

## Objetivo del Laboratorio
Este laboratorio cubre la implementación completa de Windows Server con Active Directory, configuración de servicios de red y políticas de grupo.

## Topología de Red
- Servidor: SRV-DC01 (192.168.10.10/24)
- Cliente: WIN10-CLIENT (192.168.10.20/24)
- Gateway: 192.168.10.1
- DNS: 127.0.0.1 (local)

## Criterios a Desarrollar
- 2.1.1 — Instalación y configuración básica
- 2.1.2 — Active Directory (dominio, OU, usuarios, grupos)
- 2.1.3 — Cliente en el dominio
- 2.1.4 — Servicios de red (DNS y DHCP)
- 2.1.5 — Políticas de grupo (GPO)
